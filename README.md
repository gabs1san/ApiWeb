# 🚀 ApiWeb

Projeto desenvolvido em ASP.NET Core com foco em arquitetura em camadas, aplicação de boas práticas e criação de APIs REST.

---

## 📌 Sobre o Projeto

A aplicação foi construída utilizando ASP.NET Core Web API com separação de responsabilidades entre:

- API
- Application
- Domain
- Infrastructure

O projeto possui estrutura organizada utilizando conceitos de:

- Clean Architecture
- Injeção de Dependência
- Repository Pattern
- Use Cases
- SOLID
- Controllers REST

---

## 🛠️ Tecnologias Utilizadas

- C#
- ASP.NET Core
- Swagger / OpenAPI
- Dependency Injection
- REST API
- .NET
- Visual Studio

---

## 📂 Estrutura do Projeto

```bash
ApiWeb/
│
├── ApiWeb                 # Camada da API
├── Application            # Regras de aplicação / Use Cases
├── Domain                 # Contratos e entidades
├── Infra.Repository       # Implementação de repositórios
```

---

## ⚙️ Funcionalidades

- Cadastro de clientes
- Estrutura REST
- Documentação automática com Swagger
- Arquitetura desacoplada
- Injeção de dependência
- Separação de responsabilidades
